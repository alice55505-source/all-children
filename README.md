# 全台專項統計產生器

一個可安裝的 PWA（漸進式網頁應用），用來彙整各召會上傳的週報，計算兒童與青職各項每週平均，並統計全台已上傳召會的總計。

## 使用方式

分為「兒童」「青職」兩個分頁，各自獨立上傳、儲存、統計。

### 兒童

1. 選擇要上傳資料的召會（共 28 個）。
2. 上傳該召會的週報 Excel 檔（`.xlsx`），需包含「週報矩陣」「摘要」分頁。
3. 點擊「解析並加入」，系統會讀取「週報矩陣」分頁中每週的「合計」列計算主日、小排的週平均；召會生活則取自「摘要」分頁 KPI 中的「期間有召會生活」人數。

### 青職

1. 選擇要上傳資料的召會。
2. 上傳該召會的週報 Excel 檔（`.xlsx`），需包含「週報網格」分頁。
3. 點擊「解析並加入」，系統會讀取「週報網格」分頁中「青職」欄位的每週數值，計算主日、家聚會（出訪＋受訪）、小排、生命讀經的週平均。

兩個分頁皆會顯示 28 個召會的上傳進度，以及已上傳召會的全台總計；資料僅儲存在瀏覽器的 `localStorage`，不會上傳到任何伺服器，重新整理頁面後仍會保留，也可用「清除全部資料」按鈕重設。

## 安裝為 App（PWA）

以 HTTPS（例如 GitHub Pages、Cloudflare Pages）部署後，桌面或手機瀏覽器可將此頁面「加入主畫面 / 安裝應用程式」，離線後仍可開啟介面。

## 本機開發

純靜態網頁，不需建置流程：

```bash
python3 -m http.server 8000
# 開啟 http://localhost:8000/index.html
```

## 目錄結構

```
index.html          主頁面（兒童／青職兩個分頁）
css/style.css        樣式（含深色模式）
js/app.js             解析與統計邏輯（共用模組工廠 + 兩套 XLSX 解析器）
js/xlsx.core.min.js   內嵌的 SheetJS 讀取 .xlsx 用（Apache-2.0，見 vendor/xlsx/LICENSE）
manifest.json         PWA manifest
sw.js                 Service Worker（離線快取）
icons/                App 圖示
```
