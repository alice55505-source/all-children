全台專項統計產生器

上傳各召會週報，自動算出各項每週平均，並彙總全台總計。

開始使用

建立一個新房間跟團隊共用資料，或輸入已有的房間代碼加入同一個房間。同一個房間的人看到、上傳的是同一份資料。

房間代碼：

1. 上傳報表 選擇召會後上傳，一次算出兒童＋青職

將檔案拖曳到這裡，或點擊選擇檔案

2. 上傳進度

已上傳召會 0 / 28

3. 各召會結果 依區域分組，區域列為該區平均
尚未上傳任何召會的資料

4. 全台總計

主日 =「週報網格」最後「兒童」小計欄 +「主日」類別的「國小」欄；召會生活、小排皆為「學齡前」+「國小」的每週平均。

2. 上傳進度

已上傳召會 0 / 28

3. 各召會結果 依區域分組，區域列為該區平均
尚未上傳任何召會的資料

4. 全台總計

主日、家聚會（出訪+受訪）、小排、生命讀經皆為「週報網格」分頁中「青職」欄位的每週平均值。

全台總計為已上傳召會之數值加總。資料儲存在這個房間專屬的雲端資料庫，只有知道房間代碼／連結的人看得到。

設定召會清單

區域名稱獨立一行，該區底下的召會每行一個並加上縮排（空格），兩個分頁（兒童／青職）共用同一份清單。
